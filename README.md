# 💰 SenangDuit

> Smart personal finance app for Malaysians, powered by Machine Learning

A Flutter-based mobile finance tracker that combines beautiful UX with real Data Science to help users — especially Malaysian young families — manage expenses, predict savings, and plan loan repayments intelligently.

## ✨ Features

### 🏠 Dashboard
- Real-time monthly budget tracking with progress visualization
- Daily average spending and month-end projections
- Top 5 spending categories breakdown
- Recent expenses overview

### 📊 ML-Powered Analytics
- **Linear regression** to predict month-end savings
- **IQR-based anomaly detection** for unusual expenses
- **Trend detection** (increasing/decreasing/stable)
- Personalized smart recommendations
- Beautiful pie, line, and bar charts

### 🏦 Multi-Loan Calculator
- Support for 6 loan types: PTPTN, Home, Car, Personal, Business, Custom
- Downpayment calculation in % or fixed RM amount
- Quick presets (5%, 10%, 15%, 20%, 30%)
- Visual amortization timeline
- Early payoff calculator with savings comparison

### 💸 Expense Management
- Add, edit, delete expenses
- 11 customizable categories
- Recurring expense tracking
- Filter by category
- Date-based grouping (Today, Yesterday, etc.)

## 🛠️ Tech Stack

- **Framework:** Flutter (Dart)
- **Database:** SQLite (sqflite)
- **Charts:** fl_chart
- **State Management:** Provider
- **Typography:** Google Fonts (Poppins)
- **Design:** Material 3

## 🇲🇾 Built for Malaysia

- Ringgit (RM) as default currency
- Malaysian-specific loan defaults (PTPTN 1% Ujrah, typical mortgage rates)
- Categories tailored to Malaysian lifestyle
- Localized date formats
